What is CI/CD?

CI/CD is a way to automate our work in software development. CI means Continuous Integration, where code is tested automatically. CD means Continuous Deployment, where the code is uploaded to the server automatically. It saves time and prevents manual errors.

My Pipeline

For this assignment, I am using GitHub Actions and GitHub Pages. When I push my HTML and CSS files to the 'main' branch, GitHub Actions runs a simple workflow. This workflow checks the files and then deploys my website so anyone can visit it.
